# Атрибуты таблицы `pfdo_programs`

Документ описывает актуальную структуру таблицы `pfdo_programs` в локальном зеркале PFDO. Таблица хранит основную карточку образовательной программы, часть полей нормализована в справочники, а исходные ответы PFDO сохраняются в JSONB-полях для трассировки и восстановления данных.

Актуально для базы `pfdo_51_mirror`: 5614 программ.

## Связанные справочники

| Поле в `pfdo_programs` | Справочник | Назначение |
|---|---|---|
| `kind` | `pfdo_program_kinds(id)` | Вид образовательной программы |
| `direction_id` | `pfdo_program_directions(id)` | Направленность программы |
| `edu_form` | `pfdo_program_education_forms(id)` | Форма обучения |
| `need_medical_certificate` | `pfdo_program_medical_certificate_requirements(id)` | Требование медицинской справки |
| `directory_level_id` | `pfdo_program_levels(id)` | Уровень сложности программы |
| `directory_program_document_id` | `pfdo_program_document_types(id)` | Вид документа по итогам обучения |
| `organization_id` | `pfdo_organizations(id)` | Организация, реализующая программу |
| `municipality_id` | `pfdo_main_municipalities(id)` | Муниципалитет программы |

## Атрибуты

| Атрибут | Тип | NULL | Значение |
|---|---|---:|---|
| `id` | `BIGINT` | Нет | Уникальный идентификатор программы в PFDO. Первичный ключ таблицы. Используется для ссылок, импорта, рекомендаций и связи с дочерними таблицами. |
| `operator_id` | `INTEGER` | Нет | Идентификатор регионального оператора PFDO. Для текущего зеркала используется оператор Мурманской области. |
| `municipality_id` | `INTEGER` | Да | Идентификатор муниципалитета из PFDO, к которому относится программа. FK на `pfdo_main_municipalities(id)`, индексируется для поиска по муниципалитету. |
| `search_name` | `TEXT` | Да | Название программы из поисковой выдачи PFDO. Используется как основное короткое название в списках и рекомендациях. |
| `full_name` | `TEXT` | Да | Полное название программы из детальной карточки PFDO (`detail.program.full_name`). |
| `short_name` | `TEXT` | Да | Краткое название программы из детальной карточки PFDO (`detail.program.short_name`). |
| `kind` | `INTEGER` | Да | Вид образовательной программы. FK на `pfdo_program_kinds(id)`. |
| `direction_id` | `INTEGER` | Да | Направленность программы. FK на `pfdo_program_directions(id)`. |
| `edu_form` | `INTEGER` | Да | Форма обучения. FK на `pfdo_program_education_forms(id)`: очная, очно-заочная, заочная. |
| `duration_year` | `INTEGER` | Да | Длительность программы в годах из структурированных данных PFDO. |
| `duration_month` | `INTEGER` | Да | Дополнительная длительность программы в месяцах из структурированных данных PFDO. |
| `age_group_min` | `INTEGER` | Да | Минимальный возраст обучающегося в месяцах. Используется для возрастной фильтрации рекомендаций. |
| `age_group_max` | `INTEGER` | Да | Максимальный возраст обучающегося в месяцах. Используется для возрастной фильтрации рекомендаций. |
| `need_medical_certificate` | `INTEGER` | Да | Требуется ли медицинская справка. FK на `pfdo_program_medical_certificate_requirements(id)`. |
| `modules_count` | `INTEGER` | Да | Количество модулей программы по данным PFDO. Детальные модули хранятся в `pfdo_program_modules`. |
| `directory_level_id` | `INTEGER` | Да | Уровень сложности программы. FK на `pfdo_program_levels(id)`. |
| `directory_program_document_id` | `INTEGER` | Да | Вид документа, который выдается по итогам программы. FK на `pfdo_program_document_types(id)`. |
| `video_link` | `TEXT` | Да | Ссылка на видео о программе, если она указана в PFDO. |
| `annotation_html` | `TEXT` | Да | HTML-аннотация программы из детальной карточки PFDO. Для текстового анализа HTML очищается в коде. |
| `task_html` | `TEXT` | Да | HTML-текст целей и задач программы из детальной карточки PFDO. Для текстового анализа HTML очищается в коде. |
| `duration_string` | `TEXT` | Да | Текстовое представление длительности из поисковой выдачи PFDO, например `7 лет`. Может быть удобнее для отображения, чем `duration_year`/`duration_month`. |
| `organization_name` | `TEXT` | Да | Денормализованное название организации из поисковой выдачи PFDO. Оставлено для обратной совместимости и отображения, основная связь с организацией идет через `organization_id`. |
| `address_name` | `TEXT` | Да | Денормализованный адрес из поисковой выдачи PFDO. Полная нормализованная адресная запись хранится в `pfdo_addresses`. |
| `all_region` | `INTEGER` | Да | Флаг PFDO из поисковой карточки: программа может относиться ко всему региону, а не только к одному муниципалитету. В текущем зеркале все значения равны `0`. |
| `enrollment` | `INTEGER` | Да | Статус доступности записи из поисковой карточки PFDO. Во фронтенде PFDO значения `1` и `2` дают активную кнопку записи, значение больше `2` означает, что записи нет. |
| `source_url` | `TEXT` | Да | Публичная ссылка на карточку программы на региональном портале PFDO. |
| `search_payload` | `JSONB` | Да | Очищенный снимок объекта программы из поисковой выдачи PFDO. Сейчас используется как резервный источник и для некоторых полей, которые еще не полностью вынесены из raw-формата. Ключи `average_score`, `distance`, `program_reviews` при импорте удаляются. |
| `detail_payload` | `JSONB` | Нет | Очищенный снимок детальной карточки программы PFDO. Используется для восстановления данных, доступа к исходным вложенным структурам и некоторых join-операций. Ключи `average_score`, `distance`, `program_reviews` при импорте удаляются. |
| `imported_at` | `TIMESTAMPTZ` | Нет | Дата и время последнего импорта или обновления записи в зеркале. |
| `program_document_url` | `TEXT` | Да | URL полного текста образовательной программы, если он найден в данных PFDO. |
| `program_document_local_path` | `TEXT` | Да | Локальный путь к скачанному документу программы. Используется пайплайном анализа документов и тем. |
| `program_document_file_url` | `TEXT` | Да | Прямая ссылка на файл документа программы, если она отличается от исходной ссылки или была извлечена отдельно. |
| `program_document_content_type` | `TEXT` | Да | MIME-тип скачанного документа программы, например PDF. |
| `program_document_file_size` | `BIGINT` | Да | Размер скачанного файла документа программы в байтах. |
| `program_document_downloaded_at` | `TIMESTAMPTZ` | Да | Дата и время успешной загрузки документа программы. |
| `program_document_download_error` | `TEXT` | Да | Текст ошибки загрузки документа программы, если скачать файл не удалось. |
| `organization_id` | `BIGINT` | Да | Идентификатор организации из детальной карточки PFDO. FK на `pfdo_organizations(id)`. В текущей базе заполнен у всех программ. |

## Значения справочников

### `pfdo_program_kinds`

| id | Название |
|---:|---|
| 1 | Дополнительная предпрофессиональная программа |
| 2 | Дополнительная общеразвивающая программа |
| 3 | Дополнительная образовательная программа спортивной подготовки |

### `pfdo_program_education_forms`

| id | Название |
|---:|---|
| 1 | Очная |
| 2 | Очно-заочная |
| 3 | Заочная |

### `pfdo_program_medical_certificate_requirements`

| id | Название |
|---:|---|
| 0 | Медицинская справка не требуется |
| 1 | Медицинская справка требуется |

### `pfdo_program_levels`

| id | Название |
|---:|---|
| 1 | Стартовый (ознакомительный). |
| 2 | Базовый |
| 3 | Продвинутый (углубленный) |
| 4 | Не применимо |
| 5 | Разноуровневая (несколько уровней в зависимости от модуля) |

### `pfdo_program_document_types`

| id | Название |
|---:|---|
| 1 | Нет |
| 2 | Сертификат о прохождении курса |
| 3 | Удостоверение о повышении квалификации |
| 4 | Свидетельство о присвоении квалификации |
| 5 | Свидетельство об освоении дополнительных предпрофессиональных программ в области искусств |
| 6 | Свидетельство об освоении дополнительных предпрофессиональных программ в области спорта |
| 7 | Свидетельство об обучении |
| 8 | Диплом выпускника художественной школы |
| 9 | Диплом выпускника музыкальной школы |
| 10 | Диплом выпускника спортивной школы |
| 11 | Удостоверение об освоении программы |

## Вынесенные связанные данные

Часть данных программы не хранится в `pfdo_programs` отдельными колонками, а вынесена в связанные таблицы:

| Таблица | Что хранит |
|---|---|
| `pfdo_program_keywords` | Справочник ключевых слов/тегов PFDO: `id`, `name`. |
| `pfdo_program_keyword_links` | Связь многие-ко-многим между программами и ключевыми словами. |
| `pfdo_program_modules` | Модули программы. |
| `pfdo_program_groups` | Доступные группы/наборы программы. Связь с программой идет через `program_id -> pfdo_programs.id`. |
| `pfdo_program_group_periods` | Периоды обучения внутри группы программы. Связь с группой идет через `group_id -> pfdo_program_groups.id`; первичный ключ составной: `group_id`, `period_hash`. |
| `pfdo_group_schedule_entries` | Расписание занятий группы и периода. Использует `group_id` и `period_hash` для привязки к периоду группы. |
| `pfdo_program_registry_entries` | Реестры программы: сертифицированные, бюджетные, платные и другие признаки. |
| `pfdo_program_activity_links` | Связь программы с видами деятельности по id. |
| `pfdo_program_activities` | Текстовые названия видов деятельности из детальной карточки. |
| `pfdo_program_project_links` | Связь программы с проектами PFDO. |
| `pfdo_program_calendar_topics` | Темы календарно-тематического плана, извлеченные из документа программы. |
| `pfdo_program_topic_*` | Нормализация, агрегация, классификация и очередь проверки тем программы. |

## Удаленные из модели атрибуты

Следующие поля больше не хранятся как атрибуты `pfdo_programs` и вычищаются из JSON payload при импорте:

| Атрибут | Причина |
|---|---|
| `average_score` | Не используется в логике и удален из импорта. |
| `distance` | Зависит от контекста поиска PFDO; для расстояний лучше считать геометрию по координатам `pfdo_addresses`. |
| `program_reviews` | Не используется в логике, не хранится внутри `pfdo_programs` и вычищается из JSON payload при импорте. |
